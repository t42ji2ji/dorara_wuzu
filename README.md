# Wuzu — 陪你說話的日記夥伴

> 一隻懂你的小動物，陪你把每天的心情好好說完。

## 核心理念

日記不該是一個人的事。

面對空白頁面，我們常常寫一句就停下來——不是沒有話想說，而是自言自語太孤單。

Wuzu 把日記變成對話。有個夥伴在聽，你只需要專注在「想說什麼」。而這些對話會被好好保存，在你低潮的時候，提醒你走過什麼、感謝過什麼。

## 產品特色

### WUZU 夥伴

WUZU 是一隻會陪你聊天的狗狗。

寫完一段，牠會搖搖尾巴回應你。不是冷冰冰的 AI 分析，而是像朋友一樣的對話。牠會記得你說過的事、你在乎的人、你經歷的起伏。

寫日記不再是自言自語，而是有個夥伴在聽。

### 立體的情緒理解

我們不用「開心」、「難過」這種扁平的標籤。

Wuzu 用心理學的 AVD 模型，從三個維度理解你的情緒：

| 維度 | 低 | 高 |
|:----:|:----:|:----:|
| 能量 | 平靜、放鬆 | 興奮、激動 |
| 感受 | 負面、低落 | 正面、愉快 |
| 掌控感 | 無力、被動 | 主導、自信 |

打開情緒象限圖，你會看到每一天的情緒落在哪個位置。點下去，就能回到那天的日記。

### 自動整理感恩

每篇日記，Wuzu 會幫你找出你感謝的人、事、物。

不用刻意去想「今天要感謝什麼」，只要好好說完你的一天，感恩自然會被整理出來。日積月累，你會看到生活中那些一直在的美好。

### 只有你看得到

你的日記用加密儲存，只有你能解開。

我們相信，真正的日記需要絕對的隱私。你可以放心寫下任何事，沒有人會看到。

### 匿名分享，溫暖連結

想讓別人看到你的故事？

開啟公開分享後，Wuzu 會用 AI 去除個人資訊，讓你的日記以匿名的方式被其他用戶看到。

更特別的是：**需要支持的人，會優先被看到。**

我們的演算法會讓情緒較低落的日記更容易被抽到，讓溫暖流向最需要的地方。

### 匿名對話

看到別人的日記，你可以留下一段話。

不用知道對方是誰，也不用透露自己是誰。只是單純地，讓一個陌生人知道：有人讀了他的故事，有人在乎。

### 回顧與分析

日記不是寫完就忘的流水帳。

在分析頁面，你可以：
- 從情緒象限圖看這週的起伏，點擊任一天回到那篇日記
- 從感恩列表看你這段時間感謝過的人事物
- 看你的連續天數和里程碑成就

當你低潮的時候，這些日記會提醒你：你經歷過什麼、你感謝過什麼、你是怎麼走過來的。

### 陪伴的動力

寫日記會獲得心點數，可以用來：
- 幫 WUZU 換上新項圈、帽子、裝扮
- 給別人的留言按愛心

連續寫日記會累積 streak，解鎖里程碑獎勵（3 天、7 天、30 天...一路到 1000 天以上）。

讓寫日記這件事，變成一個想持續下去的習慣。

## 設計原則

1. **對話優先**：用聊天取代書寫的壓力
2. **立體情緒**：用三維模型取代扁平標籤
3. **隱私第一**：端對端加密，絕不妥協
4. **溫暖連結**：匿名但真誠的互動
5. **回頭有用**：日記是未來的你最需要的東西
